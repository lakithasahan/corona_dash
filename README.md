# corona_dash
Live covid-19 dashboard

This is a realtime dashboard application for Covid-19 build using Django.
Realtime data obtain using Webscraping from a reliable goverment site.
Features-Covid 19 growth prediction,Target high risk age group isolation, Provide covid 19 prevention instructions directly from WHO
